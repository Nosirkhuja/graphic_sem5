Хомидов Носирхуджа Хомидович, 301 группа

Реализованные эффекты:

1) CSG объект двух сфер с кубом, операции вычитания и пересечения      3 балла

2) Нетривиальные морфинг и анимация с изменением связности объектов (коническая спираль и бутылка клейна в виде восьмерки)    3 балла
Полупрозрачные объекты с общим алгоритмом сортировки граней             3 балла
(Морфинг делается для полупрозрачного объекта)

3) NURBS поверхность, составляющая существенную часть сцены             3 балла

4) Полупрозрачные billboard с упорядоченным выводом                     2 балла

Итого: 14 баллов.

Управление:
w, a, s, d  -  Движение сферической камеры.
q, e - Прближение/отдаление
Esc - Выход

Написано на VS2010 на Windows 7. Собирать проект с помощью v100 тулсета.

В папке Release/mash.exe лежит скомпилированный бинарник.
